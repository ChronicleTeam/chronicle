# Chronicle

## Technologies

- Backend
  - Axum
  - Sqlx
- Frontend
  - SvelteKit
  - Tailwind


## Setup

### Install rust:

https://www.rust-lang.org/tools/install


### Install nodejs

https://nodejs.org/en/download

### Run database

```
docker compose up
```

### Run backend

```
cd backend
cargo run
```

### Run frontend

```
cd frontend
npm run dev
```
