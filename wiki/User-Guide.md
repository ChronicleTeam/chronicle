
# **User Guide**  

## **System Overview**  
This guide provides step-by-step instructions for accessing and utilizing key features of the system.  

### **1. Authentication**  
#### **Logging In**  
- Enter your **username** and **password** in the designated fields.  
- Click **Login** to access the system.  

---  

### **2. Data Management**  
The data management menu can be found by clicking the menu (☰) button on the top-left corner of the app, and selecting **Data Management**. 

#### **Creating a Table**  
1. Click **Add Table**.  
2. Enter a **name** for your new table **or** import an existing table.  
3. Click **Create** or **Import** to finalize.  

#### **Adding a Field**  
1. Select the desired table.  
2. Click **Edit**.  
3. Click **Add Field**.  
4. Configure the field by specifying:  
   - **Name**  
   - **Data Type**  
   - **Pertinence** (if applicable)  
5. Click **Save**, then confirm.  

#### **Adding a Row**  
1. Click **Add Row**.  
2. Enter the required data.  
3. Click **Save** to confirm. 

#### **Editing a Row**  
1. Double-click on the row you wish to edit.  
2. Enter the required data.  
3. Click **Save** to confirm.

#### **Deleting a Row**  
1. Double-click on the row you wish to delete.  
2. Click **Delete Entry**.
3. Click **Confirm Delete** to confirm.

#### **Exporting a Table**  
1. Click **Export**.  
2. Select your preferred format from the list below of currently supported formats:  
   - **CSV**  
   - **Excel Spreadsheet**  

---  

### **3. Dashboards**
The dashboards menu can be found by clicking the menu (☰) button on the top-left corner of the app, and selecting **Dashboards**.

#### **Creating a Dashboard**  
1. Click **Add Dashboard**.  
2. Enter a name.  
3. Click **Create**.  

#### **Adding Charts**  
1. Select the target dashboard.  
2. Click **Edit**.  
3. Click **+ (Add Chart)**.  
4. Configure the chart by specifying:  
   - **Name**  
   - **Chart Type**  
   - **Source Table**  
5. Click **Save** to apply changes.  

#### **Configuring Chart Axes**  
1. Click **Edit** on the desired chart.  
2. Click **Add Axis**.  
3. Define the axis properties:  
   - **Field**  
   - **Axis Type**  
   - **Aggregation Method** (if applicable)  
4. Click **Save** to confirm.  

--- 

### **4. Access Management**
**Note:** The access management modal is only available to the owner(s) of the respective resource.

The access management modal is found by clicking the **Share** button while viewing a table or dashboard

#### **Giving a user access**  
1. Enter a valid username in the username field at the bottom of the modal.  
2. Select a role for the user in the dropdown. There are three roles available:
   - Owner: Full control over table
   - Editor: Add, Modify, and Delete rows, but table columns cannot be changed
   - Viewer: Cannot modify table in anyway, but may still view and export
3. Click **Add** to give the user access

#### **Changing user access**  
1. Find the user in the user list whose permissions you want to change
2. On the right-hand side of that list item, click on the button in the **Change to:** button cluster corresponding to the role you wish the user to have.
3. **Alternatively**, if you wish to completely revoke access from the user, select the red **X** button on the right-hand side of the user to do so. This user may be given access again later on using the same steps outlined above.

--- 

### **4. User Management**
**Note:** The user management menu is only available to Admins.

The user management menu can be found by clicking the menu (☰) button on the top-left corner of the app, and selecting **User Management**. 


#### **Creating a User**  
1. Enter a valid username in the username field of the **Create User** box at the top of the menu.  
2. Enter a valid password in the password field of the same box.
3. Click **Create** to create the user.

#### **Editing a User**  
1. Find the user you wish to edit in the user list
2. Click **Edit** on that user's entry in the list.
3. Change the email and/or password of the user
4. Click **Save** to save the changes

#### **Deleting a User**  
1. Find the user you wish to edit in the user list
2. Click **Delete** on that user's entry in the list.
3. Click **Ok** in the confirmation popup to confirm
